ℹ️ About Project

Smart Hostel Food Management System

🎯 Project Objective

The Smart Hostel Food Management System is designed to predict daily food requirements using Machine Learning. It helps reduce food waste by estimating the number of students present in the hostel and calculating the required food quantity.

✨ Main Features

  • 🍽 Food Quantity Prediction
  • 🤖 Machine Learning Attendance Prediction
  • ♻️ Food Waste Calculator
  • 📊 Dashboard
  • 📈 Food Waste Analysis Charts
  • 📄 PDF Report Generation
  • 👤 User Registration
  • 🔐 Admin Login
  • 📜 Prediction History
  • 📝 Feedback System

🛠 Technologies Used

  • Python
  • Flask
  • SQLite
  • Pandas
  • Scikit-learn
  • HTML
  • CSS
  • JavaScript
  • Chart.js

📈 Benefits

  • Reduce Food Waste
  • Save Hostel Expenses
  • Improve Food Planning
  • Better Resource Management
  • Machine Learning Based Prediction
